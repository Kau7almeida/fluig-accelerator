# Snippets

Código validado e reutilizável organizado por tipo.

## Como contribuir

Toda vez que gerar um código bom com IA e validar que funciona no Fluig:

1. Salve nesta pasta na categoria correta
2. Use nome descritivo: `consulta-nf-com-filtro-periodo.js`
3. Adicione um comentário no topo explicando o que faz

## Categorias

- **formularios/** — Componentes de formulário prontos (máscaras, validações, combos dinâmicos)
- **widgets/** — Widgets reutilizáveis (tabelas, filtros, modais)
- **datasets/** — Datasets validados (REST, SOAP, JDBC, formulário)
- **eventos/** — Eventos de processo comuns (validações, decisões automáticas)
- **integracoes/** — Integrações REST/SOAP prontas
