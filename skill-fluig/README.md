# Skill Fluig

A Skill personalizada está configurada no projeto "Especialista Fluig" do Claude.

Este diretório serve como backup e referência da Skill.

## Como atualizar a Skill

1. Edite o conteúdo da Skill no projeto do Claude
2. Salve uma cópia atualizada aqui como backup

## Referências TDN

A Skill foi construída com base na documentação oficial da TDN (TOTVS Developer Network).
Quando a Skill não cobrir um tema específico, o Claude está instruído a consultar a TDN ao vivo.
